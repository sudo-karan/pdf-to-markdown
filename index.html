🔒 Your data & privacy — what happens to your file
  • Your PDF never leaves your device. It is read into memory and converted entirely in your browser — there is no upload, no server, and no database.
  • Nothing is stored. Files are not written to disk, cookies, or local storage. Close the tab and everything is gone. The only file saved is the Markdown/ZIP you choose to download.
  • No third parties. A strict Content-Security-Policy blocks every external connection — no analytics, no CDNs, no trackers. Verify in DevTools → Network: zero external requests.
  • OCR runs locally too. The OCR engine and language model are bundled and run in your browser; image data is never sent anywhere.
  • Hosting note: when opened from a website (e.g. GitHub Pages), your browser downloads this app's own files from that host, like any web page — but your PDF is never sent there. For a fully offline run with no network at all, download the project and open it locally.
